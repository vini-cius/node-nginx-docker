# Node.js Nginx Webserver with Websocket and Docker

**Used Ports** (If you want, change in .env)
 - 80 (Nginx)
 - 3000 (App Node)

## Deploying / Running
 - Run `docker-compose up`
 - GET http://localhost/home 

- Open src/index.html to see websocket implementation
